# Blog Image Prompt Agent

Generate ready-to-use image prompts for a finished blog draft — one for the cover, several inline, and a social share. The agent reads the draft, picks the right moments for visuals, and writes vivid, on-brand prompts you can paste straight into an image generator, each paired with a short rationale that explains its placement.

## Capabilities

- Read a finished blog draft and propose where images belong
- Write a vivid prompt for each placement — cover, inline, or social
- Explain the rationale behind every prompt for quick review
- Match brand voice and visual style to keep imagery on-message
- Hand off the prompts to your favorite image generator in seconds
